## **5.1 복잡도**

**5.1.1 시간 복잡도** 

- 문제를 해결하는 데 걸리는 시간과 입력의 함수 관계
- 대게 시간 복잡도는 반복문이 좌우함
- 입력 범위 n을 기준으로 로직이 몇 번 반복되는지
- 빅오 표기법: 알고리즘 최악의 실행 시간을 표기, 가장 일반적으로 많이 사용
- 10n^2 + n ⇒ O(n^2)
- *O(1) < O(log N) < O(N) < O(N log N) < O(N^2) < O(2^N) < O(N!)*
<br/>
    

**5.1.2 공간 복잡도**

- 프로그램을 실행시켰을 때 필요로 하는 자원 공간의 양
- 정적 뿐 아니라 동적인 공간도 포함
